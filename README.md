# Advanced QR Code Studio

A professional, feature-rich QR code generator with customization, batch processing, and analytics.

## Features

✨ **Basic Generator** - Quick QR code creation  
🎨 **Advanced Options** - Custom colors, sizes, error correction  
📦 **Batch Generator** - Create multiple QR codes at once  
📇 **Smart Templates** - Website, Contact, WiFi, Events, Phone, SMS  
📊 **Analytics** - Track generation statistics  
📜 **History** - Quick access to recent codes  

## How to Use

1. Open the website
2. Choose a tab (Basic, Advanced, Batch, or Templates)
3. Enter your data
4. Generate and download QR codes

## Features in Detail

### Basic Generator
- Simple, fast QR code generation
- PNG download

### Advanced Customization
- Custom dark/light colors
- Adjustable size (150-500px)
- Error correction levels (7-30%)
- Real-time preview

### Batch Processing
- Generate multiple QR codes from a list
- One per line input
- Individual downloads

### Smart Templates
- Website URLs
- Contact Cards (vCard format)
- WiFi Networks
- Event Tickets
- Phone Numbers
- SMS Messages

### Built-in Analytics
- Total generated counter
- Download tracker
- Average size tracking
- Session duration
- All data stored locally

### Smart History
- Last 20 QR codes
- Copy to clipboard
- Timestamps
- Clear history option

## Technology

- Pure HTML/CSS/JavaScript
- QR Code library from CDN
- No dependencies to install
- Works offline
- Local storage for data

## Privacy

All data is stored locally in your browser. Nothing is sent to servers.
